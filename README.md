This project is a study implementation of a Goal-Oriented Action Planning (GOAP) system using Unity. The goal was to simulate autonomous agents making decisions based on dynamic world states and objectives.

![goap (1)](https://github.com/user-attachments/assets/c4b78b06-4e91-45d8-a943-9499721dbca7)

In this simulation, AI-controlled testers enter a testing room filled with computers. Each agent independently plans and executes a sequence of actions to achieve its goal:

Find an available PC

Use the PC to test a game

Leave a review after testing

Exit the room

The system demonstrates how GOAP can be used to create flexible and scalable AI behavior, where agents adapt to the environment (e.g., occupied PCs) and generate action plans at runtime instead of relying on predefined state machines.

This project focuses on:

AI decision-making using GOAP

Dynamic action planning and execution

Resource management (shared PCs)

Emergent behavior from simple rules
